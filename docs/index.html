Fast answers

What is Matrix Scroll?

Matrix Scroll is signed commit-time provenance for agent-assisted Git. It secures Git by attaching an Ed25519-signed commit envelope that records who or what signed a change, which tool produced it, and an optional bounded scope, then lets reviewers verify that proof offline before merge.

Hardware vs emulated

Emulated mode ships now and stores the signing key locally so teams can trial the full flow immediately. The SE050 hardware path keeps the same envelope schema and verifier contract, but moves the private key into the secure element as a preview trust upgrade.

CI/CD integration

Install the SDK and hook, publish commit envelopes to refs/notes/matrixscroll, then run SSX360/matrixscroll-verify-action@v1 in GitHub Actions to verify the full PR range alongside your existing scanners and branch rules.
