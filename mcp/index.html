Exposed Provenance Verbs

6 Core Attestation & Verification Tools

We have dropped generic advisors and ecosystem search tools to focus 100% on the core security loop. The server exposes only the following attestation actions:

create_envelope

Build, sign, and save a cryptographic commit envelope for staged changes or an existing commit in the workspace using Ed25519 (actor=agent).

verify_envelope

Verify the cryptographic signature and integrity of a signed commit envelope for a specific commit SHA in the workspace.

verify_pr_range

Verify every commit envelope in a given PR or commit range (base..head) offline against validation policies.

envelope_publish_notes

Publish local signed commit envelopes to git notes (refs/notes/matrixscroll) for transport and remote verification.

status

Retrieve the active cryptographic identity provider status, device ID, public key, and mode.

audit_export

Export local commit envelopes for a given range in the workspace into a deterministic bundle directory for audit logs.

Secure Agent Enforce Gates in CI

While the MCP server signs commits locally inside VS Code or Cursor using emulated (L1) keys for free, team repositories require establishing trust. Upgrade to the Hosted Identity Authority to link agent-signed commits to verified organizational trust anchors and pass CI PR gates.

Enable Self-Attestation in Your Editor

Add the following configurations to enable the matrixscroll-mcp server in your workspace or global editor setup.

Cursor

.cursor/mcp.json

project configcursor
{
  "mcpServers": {
    "matrixscroll-mcp": {
      "type": "stdio",
      "command": "python",
      "args": ["-m", "matrixscroll.mcp"],
      "env": {
        "COPILOT_WORKSPACE": "${workspaceFolder}"
      }
    }
  }
}

Claude Desktop

claude_desktop_config.json

global configclaude desktop
{
  "mcpServers": {
    "matrixscroll-mcp": {
      "type": "stdio",
      "command": "python",
      "args": ["-m", "matrixscroll.mcp"],
      "env": {
        "COPILOT_WORKSPACE": "/path/to/target-repo"
      }
    }
  }
}
