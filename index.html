Where it fits

Sigstore and SLSA answer what was built in CI. Matrix Scroll answers who signed the commit before push. Keep both. Scanners and branch protection catch code and policy issues. Build attestations sign outputs later in the chain. Matrix Scroll covers the one gap they leave: cryptographic authorship at commit time, before the merge.

Install — the agent signs its own commits

$ pip install "matrixscroll[mcp]"
$ python -m matrixscroll.mcp

Add the server to Cursor, VS Code, or Claude Desktop and the agent signs its own commits in-loop — provenance verbs only. Full editor config and the tool surface: MCP install on GitHub →

Enforce — gate it in CI

- uses: SSX360/matrixscroll-verify-action@v1
  with:
    head-ref: ${{ github.event.pull_request.head.sha }}
    base-ref: ${{ github.event.pull_request.base.sha }}
    source: notes
